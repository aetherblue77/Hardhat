# Hardhat Simple Storage

This project is a basic Solidity + Hardhat setup that demonstrates how to deploy, interact with, and test a simple smart contract. It follows the typical Web3 development workflow and is designed for beginners who want to understand the fundamentals of smart contracts and deployment using Hardhat.
